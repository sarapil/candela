## يعمل مع ERPNext الحالي لديك

Candela لا يستبدل ERPNext — بل **يُوسّعه**. ثبّته بجانب إعدادك الحالي:

```bash
bench get-app https://github.com/sarapil/candela
bench --site your-site install-app candela
```

كل بياناتك وسير عملك وتخصيصاتك الحالية تبقى كما هي.
